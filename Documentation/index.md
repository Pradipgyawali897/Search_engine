# The Pernox Kernel Documentation

This is the top level of the Pernox kernel's documentation tree. Pernox documentation, like the engine itself, is very much a work in progress; that is especially true as we work to integrate our varied documentation formats.

## User-oriented documentation

The following manuals are written for users of Pernox.

* [Admin Guide](admin-guide/index.md): Booting and configuring the Pernox kernels.
* [User Space API](userspace-api.md): Interacting with Pernox from external applications.

## Developer-oriented documentation

The following manuals are written for developers who want to modify or extend the Pernox kernel.

* [Spyder Subsystem](spyder/index.md): Deep dive into the web traversal engine.
* [Indexer Subsystem](indexer/index.md): Understanding the term frequency and storage logic.
* [Searcher Subsystem](searcher/index.md): Architecture of the query lookup engine.
* [Build System](dev-tools/build.md): Guide to the Cargo workspace and compilation flags.
