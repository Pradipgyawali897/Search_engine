## WHAT IS PERNOX?

Pernox is a high-performance, modular search engine kernel written in Rust. It
is designed to be a lightweight, distributed traversal and indexing suite
optimized for modern hardware.

The kernel is divided into several primary subsystems:

  - Spyder: Discovery of web assets and enforcement of robots.txt.
  - Indexer: Content parsing, term frequency computation, and automatic URL tracking/discovery.
  - Searcher: Optimized query lookup engine.
  - User Space (App): The CLI gateway for user interaction with live link detection.

  - Documentation/spyder/, indexer/, searcher/: Subsystem deep-dives.

## COMPILING THE KERNEL

Ensure you have the latest Rust toolchain installed (Edition 2024).

To compile the entire suite in release mode:

    cargo build --release

## RUNNING THE ENGINE

The kernel requires a `seeds.txt` file in the root directory to begin the
initial lookup phase.

    cargo run -p app

The application layer is now split into focused modules under
`crates/app/src/`:

  - `config.rs`: runtime path configuration.
  - `runner.rs`: orchestration of preload, crawl, index, and persistence.
  - `main.rs`: thin Tokio entry point.

Runtime paths can be overridden without code changes:

    PERNOX_SEED_FILE=custom-seeds.txt PERNOX_INDEX_PATH=custom-index.json cargo run -p app
